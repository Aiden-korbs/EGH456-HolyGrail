
# RS-485

RS-485, discussed in [[Inter-Thread Communication]], highlights:

- Differential signaling
- Long cable lengths (up to 1200m)
- High speeds (up to 10Mbps)

Wiring diagram:
```
Device A             Device B
| A+ -------> A+ |
| B- -------> B- |
| GND -------- GND |
```
Comparison to legacy standard: [[RS-232]]
